<h1>ATS resume scanner</h1>

<h2>Description</h2>
This project is my attempt to create my own resume scanner which will read pdf files from a specific location, scan it for relevant keywords and inform the user whether his resume is shortlisted or not.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Python</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
